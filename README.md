# 単語によるjaccard係数を求めるpythonコード(ファイルから)
---
条件に沿ったテキストファイルを読み込んで、リストとリストのjaccard係数を求めるプログラムです。

## 係数を求めるためのtxtファイルの条件:
* 1つ1つの単語が半角スペースで区切られている
* 1つ1つの単語のリスト同士の間が一行空いている
<br>
example:

>
> int main ( int argc , char \*argv[] )
>
> int main ( void )
>
> public static void main ( String[] args )
>
